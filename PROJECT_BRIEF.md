# Project Brief

## One-liner

Summify is an AI-powered web platform that turns long-form meetings, documents, videos, podcasts, and study materials into clear, structured, actionable summaries.

## v1 Scope IN

- AI-powered summarization for supported content and document workflows.
- Fast, responsive Next.js web experience with accessible UI patterns.
- SEO-ready public pages, blog/content architecture, metadata, sitemap, and Open Graph support.
- Auth, storage, and AI provider integrations already present in the codebase.
- Coordinator-driven development using repo-local agent and skill sets.

## v1 Scope OUT

- Native mobile apps unless explicitly requested.
- Store submission flows unless a future task names them.
- New paid third-party services or production credential changes without explicit user approval.

## Monetization

Freemium / Premium SaaS. Pricing, paywall, and conversion work should route through product, growth, and guardian review.

## Agent Model

Koordinatör is the single point of contact and routes tasks through `docs/agent/COORDINATOR.md` and `docs/agent/TEAM_ROSTER.md`.
